Car Scatterplot

GitHub Repo: https://github.com/18epedersen/car-scatterplot

Course: Info 190-2 (Interactive Data Visualization) @ UC Berkeley

Date: Fall 2017

Software: Javascript, D3

Created a scatterplot using automobile data. Initially the x-scale is weight and the y-scale is horsepower. After 3 seconds, the scatterplots transitions so that the x-scale is mpg and the y-scale is acceleration.
