Secure File Sharing System

Course: CS161 (Computer Security) @ UC Berkeley

Date: Spring 2017

Partner: Ya-An H.

Software: Python

Implemented a scheme to securely store files on a malicious server & used merkle trees to efficiently store data on the server.

Used symmetric & asymmetric encryption, and MAC to achieve these properties:

  • Secure downloads
  • Secure uploads
  • Sharing Files between users
  • Revocating access to users
  • Sharing Files between users
  • Efficient updates

Some methods used:

  • get_random_bytes()
  • asymmetric_encrypt()
  • symmetric_encrypt()
  • message_authenication_code()
