Rudrata Cycle Approximation Algorithm

Course: CS170 (Efficient Algorithms & Intractable Problems) @ UC Berkeley

Date: Fall 2016

Teammates: Caroline K., Ruihan Z., & Courtney P.

Software: Python

Designed and implemented a greedy, approximation algorithm to find a cycle in a graph that touches all the vertices at most once, or returns a path that includes most of the vertices if no such path exists.

Implemented methods such as:

  • heaviestFirstSearch()
  • lookAhead()
  • findBestTeams()
